# Claude Code Starter Pack

这个 pack 适合的是 message/runtime bridge，不是 Claude Code 壳本身。

## 当前诚实边界

- `partial`
- thin message/runtime bridge
- fail-closed compatibility helper

## 当前不该吹的东西

- full terminal parity
- approval UI parity
- tool parity
- MCP parity

## 运行

```bash
pnpm run start:service-local
pnpm run starter-pack:claude-code
```
