# Runtime Diagnostics Pack

这个 pack 适合的是只读 provider triage。

## 当前诚实边界

- `partial`
- read-only diagnostics
- no invoke/write actions

## 当前不该吹的东西

- browser automation
- acquisition write
- execution-brain parity

## 运行

```bash
pnpm run start:service-local
pnpm run starter-pack:runtime-diagnostics-pack
```
