# Switchyard Stitch Surface Notes

This file is a thin public note for generated or stitched design assets.

Generated design output must follow:

1. `DESIGN.md`
2. `design-system/MASTER.md`
3. the matching page master for the governed surface

## Scope

Stitched assets may help with:

- typography and spacing consistency
- token reuse
- public-docs readability
- runtime-shell polish within the approved surface family

Stitched assets must not:

- invent a new donor mix
- reopen product identity
- override governed page masters

If a stitched asset needs a deeper design review packet, that review belongs in
a maintainer-only shelf outside the public repo plane.
