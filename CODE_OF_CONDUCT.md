# Switchyard Code of Conduct

This project expects respectful, truth-first collaboration.

更直白一点说：

- 对人保持尊重
- 对事实保持诚实
- 对分歧保持克制

## Expected Behavior

- 用建设性的方式讨论代码、文档和验证证据
- 承认不确定性，不把猜测包装成结论
- 在 review、issue、discussion 里聚焦问题本身
- 给新人留下可继续接手的上下文

## Unacceptable Behavior

- 人身攻击、骚扰、歧视或羞辱
- 故意发布误导性结论、假证据或带攻击性的“赢辩论”式交流
- 泄露他人敏感信息或凭证材料
- 把 private/manual-only information 当成 public claim 对外扩散

## Reporting

如果出现需要私下处理的情况，请直接联系当前仓库的维护者，而不是在公开线程里继续升级冲突。

## Enforcement

维护者可以删除不当内容、关闭讨论，或在必要时限制参与权限，以保护项目协作环境。
