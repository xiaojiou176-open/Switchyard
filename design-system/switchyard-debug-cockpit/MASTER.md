## Surface Identity

This page is a **read-only diagnosis bench**.

It is not:

- a hosted control plane
- a KPI dashboard
- a review shell for external stakeholders

## Donor Mapping

- Primary donor: `Linear`
- Allowed secondary donor: `Raycast` utility chrome only
- Forbidden primary donor: `Mintlify`

## Page-Specific Rules

- first row explains three truths:
  - local materials
  - current browser
  - next diagnostic action
- evidence stacks come after the summary layer
- raw JSON links and ladders are reference surfaces, not decorative widgets
- read-only state must stay visible

## Visual Constraints

- dark diagnostic shell
- dense but calm information grouping
- border-led hierarchy over glossy elevation
- no fake observability charts
- no enterprise cockpit theater
