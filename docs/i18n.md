# Switchyard i18n and Bilingual Frontdoor

## Status

- `partial`

## What This Means

`Switchyard` 现在不是一个完整多语言产品站点。  
但它已经开始把最重要的开发者入口做成 **中文可读 + 英文可搜** 的形态。

## Current Bilingual Surfaces

- `README.md`
- `docs/README.md`
- `docs/glossary.md`
- `docs/faq.md`
- `docs/api/*`
- `docs/compat/*`
- `docs/shared-provider-runtime.md`

## Why This Matters

很多仓会卡在一种很尴尬的状态：

- 中文用户看得懂，但英文搜索完全抓不到
- 英文关键词铺了很多，但中文用户看不懂真正边界

`Switchyard` 当前选择的是更克制的一种做法：

> 不装作“全站双语已完成”，  
> 但把最重要的 frontdoor 做成双语可理解。

## Truthful Status

### Claimable now

- `bilingual developer frontdoor`
- `Chinese + English key-term discoverability`

### Not claimable now

- `full product i18n`
- `fully localized website`
- `fully translated docs corpus`

## Next Reasonable Step

如果后续还要继续推：

- 补更多关键页面的英文小结
- 把 API / compat / MCP 页继续做成更稳定的双语结构
- 不要为了“看起来国际化”而写大量没人维护的半成品翻译
